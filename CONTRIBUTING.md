# Contributing

Thank you for your interest in EduPlatform Starter!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/eduplatform-starter.git`
3. Follow the [setup guide](docs/setup.md) to configure your environment
4. Create a branch: `git checkout -b feature/your-feature`

## Development

```bash
npm install
npm run dev     # Start dev server at localhost:4321
npm run build   # Verify build passes
```

## What We're Looking For

- Bug fixes and improvements to existing functionality
- New demo pages or components
- Documentation improvements
- Schema extensions (see [extending guide](docs/extending.md))
- Translations (i18n support)
- Accessibility improvements

## Pull Request Guidelines

- One feature/fix per PR
- Include a clear description of what changed and why
- Ensure `npm run build` passes
- Follow existing code style (TypeScript, Preact hooks)

## Architecture

See [docs/architecture.md](docs/architecture.md) for schema design, security model, and module overview.
